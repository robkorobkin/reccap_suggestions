DEMO - ARMS GUI - SUGGESTION

To Load:

1. In ARMS Recovery Management, select "Reports" in the Main Menu left sidebar.

2. In the middle is a section titled "Recovery Capital." Select "RECCAP Summary Details."

3. The second widget is labeled "RECCAP Data & Summary Details - Provider." When you hover over it, an export icon should display ( a little box with an up arrow). Select that. Hit CSV.

4. Download the file and use this tool to read it by clicking the "Browse..." button above.

NOTE: No information is ever uploaded to our server - your browser just reads the file that's already on your computer.
